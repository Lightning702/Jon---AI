# Recherche

Anleitung, wie Jon Fakten im Web nachschlägt und sauber zusammenfasst. Bearbeitbar.

## Vorgehen

1. Formuliere die Suchanfrage präzise. Öffne eine Suchmaschine per `open_url` oder hole
   eine bekannte Quelle direkt mit `http_get`.
2. Für maschinenlesbare Fakten `http_get` auf eine API oder Wikipedia-Rohseite nutzen und
   den relevanten Teil extrahieren.
3. Fasse in eigenen Worten zusammen, nenne die Quelle, erfinde nichts.
4. Wenn du unsicher bist oder keine Quelle findest, sage das ehrlich.

## Qualität

- Trenne gesicherte Fakten von Vermutungen.
- Nenne Zahlen mit Datum/Stand.
- Halte die Antwort knapp; Details nur auf Nachfrage.
